# camping-analysis


## 相關程式碼
-  ### 爬取評論
    - 路徑：code \\ webcrawling
    - 說明：爬取google上傳統露營以及豪華露營點的評論，包括評論者名稱、評論內容、發布時間、星等
- ### 訓練分類模型
    - 路徑：code \\ model_train_3
    - 說明：使用五種深度學習模型，分別對未平衡之原始資料集、利用大型語言模型增生評論之平衡資料集訓練分類情緒模型
    - main.py：整合各模型一起訓練(各別訓練的程式碼在同資料夾其他 python 檔案)
